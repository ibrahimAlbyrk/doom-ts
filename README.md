# DOOM-TS

A DOOM-style first-person shooter for the web: a **Canvas 2D raycaster** built with
**TypeScript + Vite**, using **Freedoom 0.13.0** art/audio.

> Foundation scaffold. `src/core` is the frozen architecture contract; most gameplay
> systems are typed stubs being filled in by parallel workers. See
> [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md).

## Quick start

```sh
npm install
npm run dev        # http://localhost:5173 — boots to the title screen
npm run build      # type-check (tsc, strict) + production bundle (vite)
npm run preview    # serve the production build
```

## Layout

- `src/core/` — **frozen contract**: shared types, Vec2/math, constants, RNG, event bus.
- `src/data/` — typed const tables (weapons, enemies, items, ammo, thing-ids, skills).
- `src/{render,world,entities,ai,weapons,combat,items,audio,input,ui,game,levels,assets}/`
  — game systems (see the ownership map in `docs/ARCHITECTURE.md`).
- `tools/extract-wad/` — Freedoom WAD → assets + manifest extractor (placeholder).
- `docs/research/` — source-of-truth research (design, engine, assets, web-arch).
